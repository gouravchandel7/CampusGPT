# CampusGPT

AI-powered admission assistant for schools, colleges and coaching institutes.

## 🚀 Features

- AI-powered chatbot for school, college and coaching websites
- Instant answers to admission-related questions
- Lead capture from student and parent conversations
- Website integration with a simple script
- Admin dashboard for managing leads and chatbot data

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Node.js
- PostgreSQL
- OpenAI API

## 📦 Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.
